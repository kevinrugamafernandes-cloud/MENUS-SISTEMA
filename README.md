# MENUS-SISTEMA

Restaurant QR ordering platform designed to modernize table ordering and internal order management.

## Features 🚀
- QR menu access by table
- Digital product catalog
- Order creation system
- Kitchen order tracking
- Cashier order management

## Technologies Used 💻
- TypeScript
- React
- Express
- PostgreSQL

## Purpose 🎯
MENUS-SISTEMA is focused on helping restaurants improve service speed and order organization through digital tools.

## Current Status 📌
Project under active development with frontend and backend structure in progress.

## Future Improvements 🔥
- Real payment integration
- Admin dashboard
- Multi-language support
- Advanced analytics
